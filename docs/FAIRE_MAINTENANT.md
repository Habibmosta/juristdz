2. Sélectionnez votre projet
3. Cliquez sur **SQL Editor**
4. Cliquez sur **New Query**
5. Ouvrez le fichier: `database/create-rpc-function-profile.sql`
6. Copiez TOUT le contenu
7. Collez dans l'éditeur SQL
8. Cliquez sur **Run**
9. Vérifiez le message "Success"

### 2️⃣ Tester (1 min)

2. Allez sur l'inscription
3. Créez un compte de test
4. Vérifiez que le modal de vérification email apparaît

Ouvrez la console (F12) et cherchez:
