## 🗄️ BASE DE DONNÉES

### Tables Créées

```sql
1. case_events          -- Timeline des événements
   ├── id
   ├── case_id
   ├── user_id
   ├── event_type       -- 'created', 'updated', 'document_added', etc.
   ├── title
   ├── description
   └── created_at

2. reminders            -- Système de rappels
   ├── id
   ├── user_id
   ├── case_id
   ├── title
   ├── reminder_date
   ├── is_completed
   └── created_at

3. calendar_events      -- Calendrier/Agenda
   ├── id
   ├── user_id
   ├── case_id
   ├── title
   ├── event_type       -- 'hearing', 'meeting', 'deadline', etc.
   ├── start_time
   ├── end_time
   └── location

4. invoices             -- Factures
   ├── id
   ├── user_id
   ├── case_id
   ├── invoice_number
   ├── invoice_date
   ├── total_amount
   └── status           -- 'pending', 'paid', 'overdue'

5. invoice_items        -- Lignes de factures
   ├── id
   ├── invoice_id
   ├── description
   ├── quantity
   ├── unit_price
   └── total

6. clients              -- Gestion des clients
   ├── id
   ├── user_id
   ├── first_name
   ├── last_name
   ├── email
   ├── phone
   └── address
```

### Sécurité

```
✅ RLS activé sur toutes les tables
✅ Policies créées pour isolation des données
✅ Chaque utilisateur voit UNIQUEMENT ses propres données
✅ Fonctions utilitaires pour génération de numéros
✅ Triggers pour updated_at automatique
```
