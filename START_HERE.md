# 🎯 COMMENCEZ ICI

## ⚡ Action Immédiate

Vous avez une erreur 401 lors de l'inscription? Voici la solution en 3 minutes:

---

## 📍 ÉTAPE 1: Exécuter le Script SQL

1. Ouvrez: https://supabase.com/dashboard
2. Allez dans **SQL Editor**
3. Ouvrez le fichier: `database/create-rpc-function-profile.sql`
4. Copiez TOUT le contenu
5. Collez dans l'éditeur SQL
6. Cliquez sur **Run**

✅ **Fait? Passez à l'étape 2**

---

## 📍 ÉTAPE 2: Tester

1. Ouvrez votre application
2. Allez sur l'inscription
3. Créez un compte de test
4. Ouvrez la console (F12)

✅ **Vous devez voir:**
```
✅ User created in auth.users: [id]
✅ Profile created successfully via RPC: {success: true, ...}
```

---

## 📍 ÉTAPE 3: Vérifier

1. Ouvrez Supabase
2. Allez dans **Table Editor** > **profiles**
3. Vérifiez que le nouveau profil existe

✅ **C'est tout! L'inscription fonctionne maintenant!**
