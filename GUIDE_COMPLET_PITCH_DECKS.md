## ⚡ OPTION RECOMMANDÉE: HTML → PDF (2 MINUTES)

### Pourquoi cette option?
- Ultra-rapide (2 minutes par deck)
- Design professionnel déjà fait
- Aucune installation nécessaire
- Gratuit

### Comment faire?

1. **Ouvrir le fichier HTML**
   - Aller dans le dossier du projet
   - Double-cliquer sur `pitch-deck-notaires.html`
   - Le fichier s'ouvre dans votre navigateur (Chrome, Firefox, Edge)

2. **Imprimer en PDF**
   - Appuyer sur `Ctrl + P` (Windows) ou `Cmd + P` (Mac)
   - Dans la fenêtre d'impression:
     * **Destination**: Choisir "Enregistrer en PDF" ou "Microsoft Print to PDF"
     * **Mise en page**: PAYSAGE (très important!)
     * **Marges**: Aucune
     * **Échelle**: 100%
   - Cliquer sur "Enregistrer"
   - Nommer: `JuristDZ_Pitch_Deck_Notaires.pdf`
   - Choisir l'emplacement (Bureau)
   - Cliquer sur "Enregistrer"

3. **Répéter pour les 2 autres**
   - `pitch-deck-avocats.html` → `JuristDZ_Pitch_Deck_Avocats.pdf`
   - `pitch-deck-huissiers.html` → `JuristDZ_Pitch_Deck_Huissiers.pdf`

**Temps total**: 6 minutes (2 min × 3 decks)
